# Tutorial: Multimodal Thinker Agent

This agent is designed to generate an image using a prompt and will compare them. Additionally, this agent requires no configuration.

  <Youtube videoId="9bQwHZy-cL0" title="SmythOS -  Multimodal Thinker Agent" />

## Agent Setup

1. In `SmythOS`, navigate to the `Templates` tab.

<Image src="/docs/img/multimodal_thinker_agent/Image 2.png" alt="alt text" />

2. You can scroll down to find the `Multimodal Thinker` Agent or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/multimodal_thinker_agent/Multimodal.png" alt="alt text" />

3. Allow the template to initialize and configure.
