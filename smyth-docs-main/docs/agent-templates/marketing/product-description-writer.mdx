# Tutorial: Product Description Writer Agent

This agent is capable of creating comprehensive and appealing product descriptions. You will provide the necessary information for the agent to craft a 150-word copy that effectively and attractively represents your product.

<Youtube videoId="JGzVwz-cn0M" title="SmythOS -  Product Description Writer Agent" />

## Product Description Agent Setup

1. Go to your SmythOS Dashboard and click on the `Create Agent` button.

<Image src="/docs/img/product_description_writer_agent/Image 1.png" alt="alt text" />

2. The template for the `Product Description Agent` is readily available. Simply select it from the drop-down menu.

<Image src="/docs/img/product_description_writer_agent/Image 2.png" alt="alt text" />

3. Click `Create` and allow the template to initialize and configure.

<Image src="/docs/img/product_description_writer_agent/Image 3.png" alt="alt text" />

## Test the Agent

Follow the steps outlined below to evaluate the `Lead Contacts Scraper` Agent.

1. Open the ChatBot embodiment and click on the chat icon.

<Image src="/docs/img/product_description_writer_agent/Image 4.png" alt="alt text" />

2. Ask the agent to write a product description.

<Image src="/docs/img/product_description_writer_agent/Image 5.png" alt="alt text" />

3. The agent will then prompt you to provide the following information for generating a product description. Providing an Image URL and specifying the language are optional. However, if no language is specified, English will be used as the default language.

<Image src="/docs/img/product_description_writer_agent/Image 6.png" alt="alt text" />

4. Let’s do an example.

>- **Product Name:** Classy Hoop Earrings
>- **Brand:** Ming’s Jewelry
>- **Colors:** Gold
>- **Target Audience:** Female aged 18 and above

5. Here’s the result.

>Presenting our Classy Hoop Earrings from Ming's Jewelry. These are not just earrings, they are a statement - a perfect blend of elegance and class. Intricately designed in a timeless hoop shape, these earrings are color-crafted in a luxurious gold that radiates a warm glow. Our Hoop Earrings are an exemplary choice for the discerning woman aged 18 and above, foreseeing fashion with a touch of class. Ming's Jewelry prides itself on creating pieces that accentuate the strength and elegance of the modern woman. With these gold hoop earrings, you can shine with confidence, knowing you're at the pinnacle of style.
