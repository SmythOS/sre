    
# Tutorial: SEO Content Writer Agent

This agent assists in creating, writing, and posting content directly to WordPress, streamlining your content management workflow.

Here is a brief overview of what the `SEO Content Writer` Agent can do:

<Youtube videoId="ClKVBqY3_t8" title="SmythOS - SEO Content Writer Agent" />

## AvesAPI Key

Before using the features that require external data, you must have an AvesAPI key. Follow these steps to get one:

1. Visit [AvesAPI's Website](https://avesapi.com/) and sign up for an account.
2. Once logged in, navigate to the API Playground via the left sidebar.
3. Copy the API key provided there.

<Image src="/docs/img/seo_content_writer_agent/Image 1.png" alt="alt text" />
## Agent Setup

To set up the SEO Content Writer Agent:

1. Go to the templates tab in SmythOS.

<Image src="/docs/img/seo_content_writer_agent/Image 2.png" alt="alt text" />
2. You can locate the `SEO Content Writer` agent by scrolling or using the search function.

<Image src="/docs/img/seo_content_writer_agent/Image 3.png" alt="alt text" />
3. Click on the `Remix` button to create your version of the agent.
4. Allow the system a few moments to initialize and configure your setup.

### Scrape Google Component Setup

To scrape data from Google SERPs, configure the component with your AvesAPI key:

1. Find and select the `Google SERP` component within your agent setup.

<Image src="/docs/img/seo_content_writer_agent/Image 4.png" alt="alt text" />
2. Enter your AvesAPI Key in the settings by clicking the gear icon on the component.

<Image src="/docs/img/seo_content_writer_agent/Image 5.png" alt="alt text" />
> **Note:** Learn more about [managing your API keys](/docs/agent-studio/key-concepts/vault?id=keys) in SmythOS.

3. Don't forget to save your settings by clicking the check icon.

### Wordpress Setup

To connect to `WordPress`:

1. In `SmythOS`, open the agent variables section.
2. Enter your WordPress username, application password, and domain.
3. Save the changes.

<Image src="/docs/img/seo_content_writer_agent/Image 7.png" alt="alt text" />
Follow these steps precisely to ensure your `SEO Content Writer` Agent operates smoothly. Happy blogging!
