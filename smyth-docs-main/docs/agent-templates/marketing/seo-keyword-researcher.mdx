# Tutorial: SEO Keyword Researcher Agent

This agent can provide users with comprehensive keyword data, including search volume, Cost Per Click (CPC), and competition levels for specific sets of keywords. To aid in devising strategies that boost web traffic, improve user engagement, and enhance overall online presence. Whether you're interested in content, industry, audience, niche, or SEO targeting, this agent will ensure you have the precise data to make informed decisions.

## Agent Setup

1. In SmythOS, navigate to the `Templates` tab.

<Image src="/docs/img/keyword_researcher_agent/Image 6.png" alt="alt text" />

1. In `SmythOS`, locate the `LLM Prompt KE API Key` component then click on the gear icon to access its settings.

<Image src="/docs/img/keyword_researcher_agent/Image 8.png" alt="alt text" />

3. You’re all set!

## Test the Agent

>**Notes to remember:**
>
> - Specify the country of your target audience because search volume and competition can vary greatly from country to country or leave blank for a global result.
> - Determine the currency you want to use. This is especially important for CPC (Cost Per Click) data. Example: usd, gbp, aud, nzd (leaving this value blank defaults to USD).
> - Provide a list of keywords, separated by comma, that you want to analyze. You could be interested in one specific keyword or a combination of several. You can list/query up to 100 keywords at a time.
> - Indicate your data source. You might have a preferred SEO tool or database from where you want to pull the data. Select either gkp or cli.
>   - gkp = shows data from Google Keyword Planner
>   - cli = shows data from Google Keyword Planner and Clickstream

1. Open the ChatBot embodiment and click on the chat icon.

2. Ask the agent to get keyword data.
