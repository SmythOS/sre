# Tutorial: Content Strategy Generator Agent

This agent creates content strategies and proposes SEO optimized content titles with meta data based on competitor domain data.

<Youtube videoId="sD2bchP-Uoo" title="SmythOS - Content Strategy Generator Agent" />

## RapidAPI Credentials

RapidAPI provides a variety of endpoints for various purposes. This agent utilizes the web-to-text converter endpoint, as well as the website analysis and SEO audit PRO.

Follow the following steps to get your Rapid API Key.

1. Go to https://rapidapi.com/
2. If you do not have an account, proceed to create one by following the registration process.
3. After completing the sign up process, you will be assigned a default API key that you can access [here](https://rapidapi.com/developer/apps).

<Image src="/docs/img/content_strategy_generator_agent/Image 1.png" alt="alt text" />

4. Copy your API key using the `copy icon`. You can also click the eye icon to see your API key. 

<Image src="/docs/img/content_strategy_generator_agent/Image 2.png" alt="alt text" />

### Website To Text Converter Subscription

1. Navigate to this [link](https://rapidapi.com/evlar-evlar-default/api/website-to-text-converter) to subscribe to the website to text converter and go to the pricing tab.

<Image src="/docs/img/content_strategy_generator_agent/Image 3.png" alt="alt text" />

2. Select a subscription plan from the options available: Basic (free), Pro , Ultra. The basic plan allows 100 API requests per month for free.

<Image src="/docs/img/content_strategy_generator_agent/Image 4.png" alt="alt text" />

3. Then, click the `Subscribe` button.

<Image src="/docs/img/content_strategy_generator_agent/Image 5.png" alt="alt text" />

### Website Analyze and SEO Audit (PRO) Subscription

1. Navigate to this [link](https://rapidapi.com/ajithjojo1230/api/website-analyze-and-seo-audit-pro/) to subscribe to the website analuyze and SEO audit and go to the pricing tab.

<Image src="/docs/img/content_strategy_generator_agent/Image 6.png" alt="alt text" />

2. Select a subscription plan from the options available: Basic (free), Pro , Ultra Plan or Mega plan. The basic plan allows 50 API requests per month for free.

<Image src="/docs/img/content_strategy_generator_agent/Image 7.png" alt="alt text" />

3. Then, click the `Subscribe` button.

<Image src="/docs/img/content_strategy_generator_agent/Image 8.png" alt="alt text" />

## Agent Setup

1. In SmythOS, navigate to the templates tab.

<Image src="/docs/img/content_strategy_generator_agent/Image 9.png" alt="alt text" />

2. You can scroll down to find the `Content Strategy Agent` or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/content_strategy_generator_agent/Image 10.png" alt="alt text" />

3. Allow the template to initialize and configure.

### Web To Text Component Setup

1. In SmythOS, look for the Web To Text API call component that has a missing key component.

<Image src="/docs/img/content_strategy_generator_agent/Image 11.png" alt="alt text" />

2. Click the gear icon to open the settings, then click the `edit` button situated in the headers section.

<Image src="/docs/img/content_strategy_generator_agent/Image 12.png" alt="alt text" />

3. Click the key icon and select your stored Rapid API Key from your vault, then click `Save` to continue.

<Image src="/docs/img/content_strategy_generator_agent/Image 13.png" alt="alt text" />

> **Note:** Click [here](/docs/agent-studio/key-concepts/vault?id=keys) to learn more about adding your API keys to the vault.

### Get Top Keywords Component Setup

1. Look for the `Get Top Keywords` API call component that has a missing key component.

<Image src="/docs/img/content_strategy_generator_agent/Image 14.png" alt="alt text" />

2. Click the gear icon to open the settings, then click the “edit” button situated in the headers section.

<Image src="/docs/img/content_strategy_generator_agent/Image 15.png" alt="alt text" />

3. Click the key icon and select your stored Rapid API Key from your vault, then click `Save` to continue.

<Image src="/docs/img/content_strategy_generator_agent/Image 16.png" alt="alt text" />

> **Note:** Click [here](/docs/agent-studio/key-concepts/vault?id=keys) to learn more about adding your API keys to the vault.

4. Don’t forget to click the check icon to save your configuration.
