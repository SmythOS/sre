# Tutorial: Backlinks Analysis Agent

This agent specializes in conducting thorough analyses of the health of backlinks within a domain or its subdomains using data from Ahrefs. It pinpoints crucial insights that users should prioritize and offers a detailed guide or practical plan to effectively tackle significant issues related to backlinking.
  
<Youtube videoId="kIGjHSSUgjg" title="SmythOS -  Backlinks Analysis Agent" />

## Ahrefs API Key

1. Go to https://app.ahrefs.com and create an account if you haven't registered yet.
2. Go to the left sidebar and select `API keys creation and management`. Proceed to account settings, then navigate to API Keys, and finally, click the `generate API Key` button.

<Image src="/docs/img/backlinks_analysis_agent/1.png" alt="alt text" />

3. Store your API Key in a secure location, or you can add it to your vault. Click [here](/docs/agent-studio/key-concepts/vault?id=keys) to learn more about adding your API keys to the vault.

## Agent Setup

1. In SmythOS, navigate to the templates tab.

<Image src="/docs/img/backlinks_analysis_agent/2.png" alt="alt text" />

2. You can scroll down to find the Backlinks Analysis Agent or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/backlinks_analysis_agent/3.png" alt="alt text" />

3. Allow the template to initialize and configure.

### Ahrefs Component Setup

1. Within the builder, find the Ahref component. Click the gear icon to access settings, then click the key button to input your Ahrefs API Key.

<Image src="/docs/img/backlinks_analysis_agent/4.png" alt="alt text" />

2. Click the key icon and select your stored Ahrefs API Key from your vault. Don’t forget to click the check icon to save your configuration.

<Image src="/docs/img/backlinks_analysis_agent/5.png" alt="alt text" />

3. You’re all set!
