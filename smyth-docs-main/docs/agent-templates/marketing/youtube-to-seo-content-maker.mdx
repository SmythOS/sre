# Tutorial: YouTube to SEO Content Maker Agent

This agent is designed to  generate content based on YouTube transcripts.

**Video Tutorial:** The create_content endpoint accepts YouTube video IDs, which are the characters found after `?v=` in the video URL (e.g., in `https://www.youtube.com/watch?v=4FT5RYuifwE`, the ID is `4FT5RYuifwE`).

## RapidAPI Credentials

RapidAPI provides a variety of endpoints for various purposes. Follow the following steps to get your Rapid API Key.

1. Go to https://rapidapi.com/
2. If you do not have an account, proceed to create one by following the registration process.
3. After completing the sign up process, you will be assigned a default API key that you can access [here](https://rapidapi.com/developer/apps). 

<Image src="/docs/img/youtube_to_seo_content_maker_agent/1.png" alt="alt text" />

4. Copy your API key using the `copy icon`. You can also click the eye icon to view your API key. 

<Image src="/docs/img/youtube_to_seo_content_maker_agent/2.png" alt="alt text" />

5. Store your API Key in a secure location, or you can add it to your vault. Click [here](/docs/agent-studio/key-concepts/vault?id=keys) to learn more about adding your API keys to the vault.

### Youtube Transcriptor Subscription

1. Navigate to this [link](https://rapidapi.com/benrhzala90/api/youtube-transcriptor) to subscribe to the Youtube Transcriptor.
2. Select a subscription plan from the options available: Basic (free) & Pro. The basic plan allows 20 API requests per month for free. 

<Image src="/docs/img/youtube_to_seo_content_maker_agent/3.png" alt="alt text" />

3. Then, click the `Subscribe` button to complete the subscription.

<Image src="/docs/img/youtube_to_seo_content_maker_agent/4.png" alt="alt text" />

## Agent Setup

1. In SmythOS, navigate to the templates tab.

<Image src="/docs/img/youtube_to_seo_content_maker_agent/5.png" alt="alt text" />

2. You can scroll down to find the `Youtube to SEO Content Maker` agent or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/youtube_to_seo_content_maker_agent/6.png" alt="alt text" />

3. Allow the template to initialize and configure. 

### Get Youtube Transcript Component

1. Click the gear icon to open the settings, then click the `edit` button situated in the headers section.

<Image src="/docs/img/youtube_to_seo_content_maker_agent/7.png" alt="alt text" />

2. Click the key icon and select your stored Rapid API Key from your vault, then click `Save` to continue.

<Image src="/docs/img/youtube_to_seo_content_maker_agent/8.png" alt="alt text" />

3. Then, don’t forget to click the check icon to save your configuration.

<Image src="/docs/img/youtube_to_seo_content_maker_agent/9.png" alt="alt text" />

1. You’re all set!
