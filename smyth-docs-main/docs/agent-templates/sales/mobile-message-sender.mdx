# Tutorial: Mobile Message Sender Agent

This agent is designed to send mobile SMS or WhatsApp messages via the Twilio API.

<Youtube videoId="j5LDedUh0_g" title="SmythOS -  Mobile Message Sender Agent" />

## Twilio Credentials

1. Signup for a Twilio account here > https://www.twilio.com/
2. Learn more [here](https://app.smythos.com/doc/#/components/integrations/twilio?id=send-sms) on how to get your Twilio send SMS credentials.
3. Learn more [here](https://app.smythos.com/doc/#/components/integrations/twilio?id=send-whatsapp-message) on how to get your Twilio WhatsApp credentials.

## Agent Setup

1. In `SmythOS`, navigate to the templates tab.

<Image src="/docs/img/mobile_message_sender_agent/Image 1.png" alt="alt text" />

2. You can scroll down to find the `Mobile Sender` Agent or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/mobile_message_sender_agent/Image 2.png" alt="alt text" />

3. Allow the template to initialize and configure. 

### Send SMS Setup

1. In `SmythOS`, locate the `Twilio Send SMS` component. Click the gear icon to open the settings.

<Image src="/docs/img/mobile_message_sender_agent/Image 3.png" alt="alt text" />

2. Click on the key icon, and choose your saved `Twilio Based64 Auth Key`. Don’t forget to click the check icon to save your configuration.

<Image src="/docs/img/mobile_message_sender_agent/Image 4.png" alt="alt text" />

### Send Whatsapp Setup

1. Locate the `Twilio Send Whatsapp` component. Click the gear icon to open the settings.

<Image src="/docs/img/mobile_message_sender_agent/Image 5.png" alt="alt text" />

2. Click on the key icon, and choose your saved `Twilio Based64 Auth Key`. Don’t forget to click the check icon to save your configuration.

<Image src="/docs/img/mobile_message_sender_agent/Image 6.png" alt="alt text" />

### LLM Prompt Component Setup

1. Locate the `LLM Prompt Twilio Credentials` component.

<Image src="/docs/img/mobile_message_sender_agent/Image 7.png" alt="alt text" />

2. Click the gear icon and add your Twilio account SID and phone numbers. Don’t forget to click the check icon to save your configuration.

<Image src="/docs/img/mobile_message_sender_agent/Image 8.png" alt="alt text" />

3. You can now start sending SMS.
