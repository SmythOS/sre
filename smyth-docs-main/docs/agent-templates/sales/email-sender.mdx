# Tutorial: Email Sender Agent

This agent is designed to help users send emails.

<Youtube videoId="BjuW0cOsbpo" title="SmythOS - Email Sender Agent" />

## Resend API Key

1. Go to resend.com and create a developer account.
2. Login, and click on "API Keys" in the right sidebar.

<Image src="/docs/img/email_sender_agent_agent/Image 1.png" alt="alt text" />

3. Next, click on the `Create API Key` button.

<Image src="/docs/img/email_sender_agent/Image 2.png" alt="alt text" />

4. Provide a unique name for your API Key that clearly identifies its purpose, then click the `add` button to proceed.

<Image src="/docs/img/email_sender_agent/Image 3.png" alt="alt text" />

5. You can only view the key once, so be sure to copy it and store it in a secure location.

<Image src="/docs/img/email_sender_agent/Image 4.png" alt="alt text" />

6. Then, register your email domain. You can only send email from email domains registered in your Resend account.

<Image src="/docs/img/email_sender_agent/domain.png" alt="alt text" />

## Agent Setup

1. In `SmythOS`, navigate to the templates tab.

<Image src="/docs/img/email_sender_agent/Image 5.png" alt="alt text" />

2. You can scroll down to find the `Email Sender` Agent or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/email_sender_agent/Image 6.png" alt="alt text" />

3. Allow the template to initialize and configure.

### Adding API Key to the Vault

Your API Key will be needed on your Send mail component.

1. Navigate to the Vault section on your dashboard.

<Image src="/docs/img/email_sender_agent/Image 7.png" alt="alt text" />

2. Click the `Add Key` button to add your Resend API Key.

<Image src="/docs/img/email_sender_agent/Image 8.png" alt="alt text" />

3. Enter the API Key generated from your Resend account. Choose a unique Key name that you can easily distinguish whenever it's needed. Select `API Call` for the Scope, then click `Done` to save your API Key.

<Image src="/docs/img/email_sender_agent/Image 9.png" alt="alt text" />

4. Your Resend API Key has been successfully added to your Vault and is now ready for use.

<Image src="/docs/img/email_sender_agent/Image 10.png" alt="alt text" />

### Resend Component Setup

1. Locate the `Send Email` component, and click on the gear icon to access its settings.

<Image src="/docs/img/email_sender_agent/Image 11.png" alt="alt text" />

2. Select the key icon to access your previously stored API key for Resend.

<Image src="/docs/img/email_sender_agent/Image 12.png" alt="alt text" />

3. Choose your Resend API key.

<Image src="/docs/img/email_sender_agent/Image 13.png" alt="alt text" />

4. Don’t forget to click the check icon to save your configuration.

<Image src="/docs/img/email_sender_agent/Image 14.png" alt="alt text" />

5. You’re all set!
