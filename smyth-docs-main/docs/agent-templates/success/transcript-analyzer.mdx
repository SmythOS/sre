# Tutorial: Transcript Analyzer Agent

<Youtube videoId="gsRJ_46GFPw" title="SmythOS -  Transcript Analyzer Agent" />

This agent exclusively analyzes meeting transcripts. Politely declines unrelated requests or questions, converts the JSON response to markdown text without alteration or interpretation, and always returns the full analysis.

This agent requires no configuration and is ready to use.

## Agent Setup

1. In SmythOS, navigate to the templates tab.

<Image src="/docs/img/transcript_analyzer_agent/1.png" alt="alt text" />

2. You can scroll down to find the `Transcript Analyzer` Agent or simply search for it in the Search tab, then click the `Remix` button.

<Image src="/docs/img/transcript_analyzer_agent/2.png" alt="alt text" />

3. Allow the template to initialize and configure.

## Agent Test

1. In the builder, toggle debug to run the agent.

<Image src="/docs/img/transcript_analyzer_agent/3.png" alt="alt text" />

2. Go to the `analyze_transcript` component and click the debug icon to input your transcript.

<Image src="/docs/img/transcript_analyzer_agent/4.png" alt="alt text" />

3. Then, paste your transcript in the editor.

<Image src="/docs/img/transcript_analyzer_agent/5.png" alt="alt text" />

4. Click the `Step` button to allow the component to process your request.

<Image src="/docs/img/transcript_analyzer_agent/6.png" alt="alt text" />

5. Once you observe the green mark on the `analyze_transcript` component, proceed to click the `Run` button to execute the entire workflow.

<Image src="/docs/img/transcript_analyzer_agent/7.png" alt="alt text" />
